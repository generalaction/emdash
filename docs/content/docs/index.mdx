Scrawl is an open source desktop app for running multiple writing agents in parallel. Each agent works in an isolated Git worktree, so they don't interfere with each other. Forked from [Emdash](https://github.com/generalaction/emdash) and repositioned for writers — copywriters, content writers, proposal writers, and technical writers.

## Capabilities

- **[Parallel agents](/parallel-agents):** Run multiple agents on the same writing brief, each in its own workspace
- **[Provider support](/providers):** Use any of 22+ CLI-based agents: Claude Code, Gemini, Qwen Code, and more
- **[Best-of-N](/best-of-n):** Run multiple agents on the same brief, compare their drafts, and pick the best
- **[Diff view](/diff-view):** Review changes across agents side-by-side
- **[Kanban view](/kanban-view):** Organize writing tasks visually across your workflow
- **[Issue integration](/issues):** Pull briefs from Linear, Jira, or GitHub Issues directly

## How It Works

Click "Add Task" to create one or more workspaces. Each workspace runs its own agent. You can then review the diff when done, iterate if needed, and open a PR inside of Scrawl.
