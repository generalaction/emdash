Scrawl is [open source](https://github.com/gidea/scrawl) and we strive to be fully transparent about what we're working on and what's next. This roadmap is a living document that we'll update as we make progress.

## Inherited from Emdash

Scrawl inherits a mature feature set from [Emdash](https://github.com/generalaction/emdash), the Agentic Development Environment we forked from. These features work today:

- Parallel agent orchestration with 22+ CLI agents
- Git worktree isolation for each task
- Diff view and inline editing
- Kanban board for task management
- Issue integration (Linear, Jira, GitHub Issues)
- MCP server management
- Skills system (Agent Skills standard)
- SSH remote development
- CI/CD check monitoring
- tmux session persistence

## Writer-Specific Features (Planned)

These are the features we're building specifically for writers:

- **Cloud vector database integration** — Connect to vector databases via MCP for retrieval-augmented generation (RAG), enabling research-backed writing workflows
- **Writer-specific skills** — Reusable prompt packages for style guide enforcement, tone matching, audience adaptation, and format compliance
- **Content project templates** — Pre-configured project setups for common writing workflows (blog posts, proposals, technical docs, marketing copy)
- **Writer-optimized UI** — Interface refinements tailored to content creation workflows rather than software development
