## Findings considered and rejected (do not re-audit)

- **Claude first-wins dedup undercounting streamed messages**: rejected by measurement —
  across the 8 largest real transcripts (18,538 message ids), zero duplicate ids carried
  differing usage. First-wins is sound.
- **Heatmap "today" staleness after midnight / DST drift in `startOfWeek` arithmetic
  (`aggregate.ts:52`)**: real but cosmetic; not worth a plan.
- **Non-atomic `saveIndex` write**: `loadIndex` self-heals from corruption; fine as is.
- **Scanner symlink handling**: `withFileTypes` skips symlinked dirs — no loop risk;
  not following them is an acceptable default.
